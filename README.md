# Retail Sales & Inventory Analysis

This project analyzes retail and warehouse sales data using Excel and Tableau to identify product performance trends, top suppliers, and inventory inefficiencies.

## Tools Used
- Excel
- Tableau

## Project Overview
The dashboard was created to explore:
- Sales performance by item type
- Relationship between warehouse and retail sales
- Top suppliers by retail sales
- Top suppliers by warehouse sales

## Key Insights
- Beer showed the highest warehouse sales volume.
- Liquor showed relatively strong retail performance compared to some other categories.
- A small number of suppliers contributed disproportionately to warehouse sales.
- The scatter plot showed that many items had low retail sales, while a few items had very high warehouse distribution.
